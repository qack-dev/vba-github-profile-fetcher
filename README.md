# GitHubユーザー情報ビューア (Excel VBA)

![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)

Microsoft Excel上で動作する、GitHubユーザー情報取得ツールです。
指定したユーザーの公開プロフィールとリポジトリの一覧を、GitHub APIを利用して取得し、見やすくシートに整形して表示します。
Geminiを用い、Vibe Codingによって作成されました。

## 主な機能

*   **簡単な操作**: Excelシート上のボタンを押し、ユーザー名を入力するだけで、誰でも簡単に情報を取得できます。
*   **詳細な情報取得**: 以下の情報を自動で取得・表示します。
    *   **ユーザー基本情報**: プロフィール、フォロワー数、所属、URLなど
    *   **公開リポジトリ一覧**: リポジトリ名、開発言語、スター数、フォーク数、説明など
*   **便利なコマンド生成**: 各リポジトリの `git clone` コマンドや、全リポジトリを一括でクローンするためのコマンドを自動で生成します。
*   **インストール不要**: `git` コマンドや専門的な環境構築は一切不要です。Excelファイル一つで完結します。

## 使い方

### 1. ツールのダウンロード

このツールは `git clone` を行う必要はありません。以下の手順でExcelファイルのみをダウンロードしてください。

1.  このリポジトリのファイル一覧から `GitHubユーザー情報ビューア.xlsm` をクリックします。
2.  次の画面の右上にある `Download raw file` ボタンをクリックすると、ファイルのダウンロードが開始されます。

### 2. 実行手順

1.  ダウンロードした `GitHubユーザー情報ビューア.xlsm` を開きます。
2.  Excelの上部に「セキュリティの警告」が表示された場合は、**「コンテンツの有効化」** または **「マクロを有効にする」** ボタンをクリックしてください。
3.  `Sheet1` に表示されている **「生成」** ボタンをクリックします。
4.  「GitHubのユーザー名を入力してください」というダイアログが表示されるので、情報を取得したいユーザー名（例: `qack-dev`）を入力し、「OK」をクリックします。
5.  処理が完了すると、自動でシートにユーザー情報とリポジトリ一覧が書き出されます。

## 前提条件

*   Microsoft Excel (マクロが実行可能なバージョン)
*   インターネット接続環境 (GitHub APIとの通信に必要です)

## 技術的な詳細

本ツールはExcel VBAのみで開発されています。

*   **API通信**: `MSXML2.XMLHTTP.6.0` を利用して、GitHub API (`https://api.github.com`) へHTTP GETリクエストを送信しています。
*   **JSON解析**: APIから返却されるJSON形式のレスポンスは、VBA標準機能だけでは扱いにくいため、`ScriptControl` オブジェクトを介してJScriptの `eval` 関数を利用し、動的に解析しています。これにより、外部ライブラリを追加することなく、単一のExcelファイルで動作を完結させています。
*   **エラーハンドリング**: APIからの応答ステータスコード（`200 OK`, `404 Not Found`, `403 Forbidden` など）を判定し、ユーザーが見つからない場合やAPIのレート制限に達した場合に、適切なメッセージを表示する機能を実装しています。

## 注意事項

*   GitHub APIには、短時間に大量のリクエストを行うと一時的にアクセスが制限される「レート制限」が設けられています。`APIのレート制限に達した可能性があります。` というエラーが表示された場合は、しばらく時間を置いてから再度お試しください。
*   本ツールで取得できるのは、GitHub上で公開されている情報のみです。非公開の情報は取得できません。
